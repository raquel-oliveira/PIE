# PIE

PIE is a Pascal-like programming language. This project consists of a compiler for PIE.

## How to compile and run

### Compiling

```bash
make
```
### Running

#### Recursive parser

```bash
./recursive pathtofile.pie
```
#### Table parser

```bash
./table pathtofile.pie
```
### Example
```bash
make
./recursive codesamples/merge_sort.pie
```
